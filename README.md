# Мастер и Маргаритка - Цветочный магазин

Сайт цветочного магазина "Мастер и Маргаритка" в Рязани.

## Описание

Одностраничный сайт для цветочного магазина с информацией о магазине, способах оформления заказа и преимуществах.

## Технологии

- Next.js
- React
- TypeScript
- Tailwind CSS

## Установка и запуск

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm dev

# Сборка для production
pnpm build

# Запуск production сервера
pnpm start
```

## Функционал

- Бегущая строка с информацией
- Информационный стенд с кнопками для связи
- Галерея работ
- Отзывы клиентов
- Блок преимуществ
